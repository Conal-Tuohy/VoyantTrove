Trove + Voyant

This web application combines three applications:

  • The text visualization tool Voyant
  • the TroveProxy application which converts Trove data into TEI XML, which can be imported into Voyant Tools
  • A Trove Query-Builder form which helps to construct a Trove query

You can now:

  • Open the Trove Query Builder form to generate a URL which identifies a set of Trove records, and specifies that they be returned in the form of a TEI corpus. Using the form you can pass the URL of that corpus to Voyant for it to import and analyse.
  • Or you can open Voyant Tools directly,
  • or create and run Spyral notebooks.
